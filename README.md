# ストック管理Labo

まとめ買いしたものがどれくらい持つか、今月何を買えばよいかを見える化するための React + TypeScript + Vite アプリです。

## できること

- 在庫の追加・編集・削除
- localStorage への保存
- 在庫状態の自動判定
  - 現在数 > 最低必要数: 安心
  - 現在数 = 最低必要数: そろそろ確認
  - 現在数 < 最低必要数: 買う候補
- 重要度フィルター
- カテゴリフィルター
- まとめ買い検証用の消費メモ

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```
